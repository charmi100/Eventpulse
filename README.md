# EventPulse

EventPulse is an AI-powered event discovery platform that visualizes events on an interactive heatmap.

## Features
- Interactive event map
- Event heatmap
- Event creation
- Personalized event recommendations
- AI event assistant
- Organizer analytics

## Tech Stack
Frontend:
Next.js
React
TailwindCSS

Backend:
Node.js
Express

Database:
MongoDB

Maps:
Mapbox
